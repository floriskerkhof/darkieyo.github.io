Differentials

A drill-down from the most basic idea: a differential is the tiny output movement caused by a tiny input movement.

Level 0

Change

Start with only this: something moves.

\(dx = \text{a tiny move in } x\)
\(dy = \text{the matching tiny move in } y\)
Level 1

Connection

A function connects input and output.

\(y=f(x)\)

If \(x\) moves, \(y\) usually moves.

Level 2

Local multiplier

The derivative is the multiplier at one exact point.

\(\frac{dy}{dx}=f'(x)\)
Core formula

The whole idea in one line

\(dy=f'(x)dx\)

tiny output move = local multiplier × tiny input move

What is \(dx\)?

The chosen input nudge. Example: move \(x\) from 3 to 3.01, so \(dx=0.01\).

What is \(f'(x)\)?

The slope right here, not globally. For \(y=x^2\), the slope is \(2x\). At \(x=3\), slope is 6.

What is \(dy\)?

The predicted output nudge from the tangent line. At \(x=3\), \(dy=6dx\).

Why is this not exactly \(\Delta y\)?

\(\Delta y\) is the true curve movement. \(dy\) is the straight-line approximation. For tiny moves, they are almost the same.
